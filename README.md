# NgSweeper
A fun practice program. MineSweeper written for evergreen browsers in AngularJS.

Written in TypeScript 1.5.

## Dependencies
* [Node](https://github.com/joyent/node)
* (Various other dependencies are downloaded)

## Installation
Type
`npm install`

## Running
Type
`npm start`

Go to the URL:

http://localhost:8000/
